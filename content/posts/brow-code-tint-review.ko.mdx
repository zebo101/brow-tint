Brow Code는 호주 출신의 전문 브로우 브랜드로, 어느새 전 세계 하이엔드 살롱에서 기본 틴트 시스템으로 자리 잡았습니다. 최근 몇 년 사이 브로우 예약을 잡았을 때 브로우 아티스트가 "Brow Code 시스템"을 언급했다면, 살롱 시술용으로 설계된 전문가 전용 틴트, 컨디셔닝 트리트먼트, 애프터케어 제품 라인 이야기를 하고 있는 것입니다.

이 리뷰는 Brow Code 틴트가 다른 전문 브로우 틴트(RefectoCil, Wunderbrow 등)와 무엇이 다른지, 일반적인 고객 경험은 어떤지, 정직한 트레이드오프, 그리고 예약 전에 Brow Code 셰이드 범위를 자신의 얼굴에 미리 보는 방법을 다룹니다.

## Brow Code 틴트의 정체

Brow Code 틴트는 라이선스가 있는 살롱과 브로우 아티스트에게만 판매되는 전문가용 반영구 염료이며, 소비자에게 직접 판매되지 않습니다. 시스템은 다음을 포함합니다:

- **컬러 범위** — 일반적으로 갈색, 에스프레소, 애쉬, 웜 톤으로 구성된 큐레이션된 셰이드 세트로, 현장에서 과도한 즉석 믹싱 없이 대부분의 자연 브로우 컬러를 매칭합니다.
- **시술 프로토콜** — 자체 타이밍 권장사항과 디벨로퍼 비율; 브로우 아티스트는 브랜드 전용 교육을 거쳐 정확한 시술법을 익힙니다.
- **헤어 + 스킨 하이브리드 틴팅** — 처방은 눈썹 모발에 컬러를 입히는 *동시에* 그 아래 피부에도 가볍게 착색하도록 설계되어, 모발 컬러링만으로는 표현되지 않는 풍성해 보이는 브로우 효과를 만듭니다.

브랜드는 브로우 라미네이션 제품, 컨디셔닝 오일, 애프터케어 제품도 판매합니다. 살롱의 "Brow Code 서비스"는 보통 아티스트가 틴트뿐만 아니라 Brow Code 풀라인을 조율된 시스템으로 사용하는 것을 의미합니다.

## 브로우 아티스트가 선호하는 이유

여러 전문 틴트 브랜드를 사용해 본 브로우 아티스트와 이야기해 보면 Brow Code에 대해 일관되게 세 가지 이야기가 나옵니다:

1. **예측 가능한 컬러 결과.** 요청한 셰이드가 그대로 나옵니다. 다른 일부 전문 틴트는 예상 컬러와 최종 컬러 사이의 편차가 커서, 굵거나 다공성인 눈썹에서는 특히 그렇습니다.
2. **합리적인 처리 시간.** 표준 시술이 5~10분 범위에 위치해, 바쁜 살롱 일정에 적합하면서도 깊은 발색까지 충분합니다.
3. **자연스럽게 페이드되는 피부 착색.** 헤어 + 스킨 하이브리드 틴트는 피부 착색이 시각적 만족 시점을 넘기지 않도록 캘리브레이션되어 있습니다 — 대부분의 고객이 7~10일에 걸쳐 피부 착색이 자연스럽게 옅어져 모발 틴트와 어우러진다고 보고합니다.

이는 시술자 레벨의 디테일이지만, 고객 경험으로 직결됩니다: 더 예측 가능한 결과, 의자에 앉아 있는 시간 단축, 시술 직후의 "방금 눈썹 염색했어요" 티가 덜 나는 기간.

## 컬러 범위와 셰이드 매칭

Brow Code의 셰이드 라인업은 망라적이라기보다는 큐레이션되어 있습니다. 가능한 모든 브로우 컬러가 있는 것은 아니며 — 자연 인체 브로우 컬러의 대부분을 커버하는 가장 자주 요청되는 8~12개 셰이드에 집중합니다.

살롱에서 가장 많이 쓰이는 Brow Code 셰이드:
- **라이트 브라운 / 블론드** — 은은한 디테일을 원하는 밝은 모발 고객용
- **미디엄 브라운 / 뉴트럴** — 대부분의 브루넷 고객 기본
- **다크 브라운 / 에스프레소** — 풀 블랙까지 가지 않으면서 또렷한 깊이를 원하는 고객용
- **애쉬 / 쿨톤 옵션** — 웜톤 시프트를 피하고 싶은 쿨톤 고객용
- **웜 / 레드톤 옵션** — 어번 또는 웜톤 베이스 눈썹 고객용

브로우 아티스트는 흔히 인접한 두 셰이드를 블렌딩하여 그 사이의 톤을 잡습니다. 숙련된 아티스트는 자연 눈썹 색, 모발 색, 피부 언더톤을 모두 본 다음에야 Brow Code 셰이드를 추천합니다 — 단순히 튜브와 눈썹을 매칭하지 않습니다.

## Brow Code 틴트 시술 가격

Brow Code는 살롱 전용 제품이므로 가격은 제품 자체보다 살롱의 포지셔닝에 더 의존합니다. Brow Code 틴트 서비스의 일반적인 가격대:

- **미국** — 틴트 단독 30~80달러; 틴트 + 라미네이션 결합 80~200달러
- **영국** — 틴트 25~60파운드; 결합 55~120파운드
- **호주** — 틴트 AUD 30~70달러 (Brow Code의 홈 마켓이라 보통 가장 저렴); 결합 AUD 80~160달러
- **유럽 본토** — 틴트 30~70유로; 결합 60~130유로

같은 살롱에서 일반 전문 브로우 틴트 대비 프리미엄은 보통 크지 않습니다(5~15달러) — 대부분의 비용은 제품 자체가 아닌 아티스트의 시간과 살롱 오버헤드입니다.

## DIY 가용성 — 사실상 없음

Brow Code는 전문가 전용 제품 라인입니다. 브랜드는 핵심 틴트 시스템을 소비자에게 직접 판매하지 않으며, 공급은 라이선스 브로우 아티스트와 살롱으로 제한됩니다. 일부 Brow Code 소비자용 제품(눈썹 컨디셔닝 오일, 애프터케어)은 리테일이나 온라인에서 찾을 수 있지만, 틴트 자체는 살롱 시술 전용입니다.

"Brow Code 틴트 집에서 살 수 있는 곳"을 검색하고 있었다면 — 이는 유통 문제가 아니라 의도된 포지셔닝입니다. 브랜드는 가정용이 아닌 전문 서비스로 자리매김되어 있습니다.

## 정직한 트레이드오프

Brow Code가 모든 면에서 대안보다 우월한 것은 아닙니다 — 특정 사용 사례에 최적화되어 있습니다. 부족한 점:

- **셰이드 깊이 제한.** 매우 어두운 아시아 고객 작업이나 특수한 울트라 쿨 애쉬용 블루 블랙은 없습니다; 이런 고객을 다루는 브로우 아티스트는 더 폭넓은 RefectoCil 라인을 쓰기도 합니다.
- **살롱 전용 접근성.** 가정용 옵션이 없다는 것은 매번 리프레시마다 예약이 필요하다는 뜻입니다. 이미 자신에게 맞는 셰이드를 아는 사람에게는 DIY 키트가 더 실용적입니다.
- **일부 살롱의 프리미엄 포지셔닝.** 일부 살롱은 Brow Code 서비스에 일반 틴트보다 상당한 마크업을 붙입니다. 예약마다 15~30달러를 더 낼 가치가 있는지는 개인의 판단입니다.

진정으로 빛나는 영역: 첫 틴트 고객, 특별한 행사 예약, 처방을 견디는 민감성 피부 고객, 그리고 시스템에 특화된 교육을 받은 브로우 아티스트로부터 예측 가능한 결과를 원하는 모두.
